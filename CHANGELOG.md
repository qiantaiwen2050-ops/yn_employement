## [v0.7.0-M7] — 2026-04-22

**M7 收尾**：操作日志全链路审计 + Apache POI Excel 导出 + 对比分析补减少数列

后端：
- `SysLogService` 嵌入 20+ 关键写操作（登录/改密/备案审核/报表审核退回修改删除/通知 CUD/调查期增改/用户角色 CUD/国家上报）
- `/system/log` 列表接口，支持按 action / username / 日期范围过滤（最多 200 条）
- Apache POI XLSX 导出：通用 `XlsxWriter`（表头加粗 + 浅蓝底 + 列自适应 + 冻结首行）
- 7 个原 CSV 导出全部升级到真 .xlsx
- `CompareRow` 补 `decreaseCountA` / `decreaseCountB`，对齐 SOW §对比分析 6 项指标

前端：
- 「系统管理 → 操作日志」页：26 种操作分类筛选 + 颜色 tag 区分
- 7 个导出按钮 "导出 CSV" → "导出 Excel"
- 对比分析表升级两层表头（A 期 / B 期 各含 企业数 / 调查期人数 / 减少数）

## [v0.6.0-M6] — 2026-04-22

**M6 系统管理 + 国家接口**：用户/角色/监控 + 通知 + 国家失业监测系统对接 Mock

后端：
- 用户管理：跨类型 CRUD + 重置密码 + 删除前校验（企业有上报数据时拒绝）
- 角色管理（RBAC）：3 个内置角色（省级/市级/企业）+ 26 个权限码（5 个分组）
- 系统监控：CPU / 内存 / JVM / 硬盘（JMX）+ 在线用户（`JwtSessionTracker`）
- 国家接口 Mock：本地模拟 ~10% 概率失败；`nation_report_log` 全链路（请求/响应/回执/错误码/重试链）
- M4 「批量上报国家」改为路由经 NationReportService → 同一份数据流 + 同一份日志

前端：
- 菜单升级支持 sub-menu 分组（业务管理 / 汇总分析 / 系统管理 / 通知与上报）
- 用户/角色/监控/通知/国家接口 5 个页面

## [v0.5.0-M5] — 2026-04-22

**M5 省级分析类**：4 种可视化分析

- 取样分析：按地区企业数 + 占比 + 排名（饼图）
- 多维分析：1–3 维（地区 / 性质 / 行业）自由组合交叉表
- 对比分析：两期对比（强制周期类型一致）双折线图 + 表格
- 趋势分析：≥3 个连续期（强制 `period_type` 一致 + `seq_in_year` 连续）多线折线图
- 通用 `EChart` 组件 + 4 个 Excel 导出（按 SRS 文件名规则）
- DataSeeder 扩充：8 家备案企业跨 5 地区 / 3 性质 / 6 行业 + 10 期 PROV_OK 报表

## [v0.4.0-M4] — 2026-04-22

**M4 省级管理类**：备案审 / 报表管 / 修改留痕 / 删除 / 退回 / 汇总

- 省级备案审核（通过 / 退回 / Excel 导出）
- 省级报表管理：跨地区筛选 + 详情（原始 vs 有效对照）+ 04→05 通过 / 04→06 退回 / 05→07 上报国家
- 数据修改：留痕到 `report_revision` 表（`before_json` 快照 + 修改原因 + 修改人/时间），原始 `report` 行不动
- 数据删除：逻辑删除 + 状态 07（已上报国家）保护
- 数据汇总：按调查期 + 维度（地区/性质/行业）聚合，使用 effective 值

## [v0.3.0-M3] — 2026-04-22

**M3 市级审核流转 + 通知**

- 市级审核：辖区数据隔离（按 `region_code`）、待审核列表、详情、单条/批量审核通过、退回（必填原因 + 常用原因模板）、上报省级
- 报表状态机扩展：01→02 / 03，02→04
- 通知 CRUD + 角色化可见性（企业/市级看 省级 + 本辖区市级；过期自动隐藏）
- **报表尝试历史 (`report_attempt`)**：每次企业提交独立留痕，退回与重报互不干扰（解决用户反馈"退回原因被覆盖"问题）

## [v0.2.0-M2] — 2026-04-22

**M2 企业端**：备案 + 月度填报 + 历史查询

- 调查期模块：一键按年生成 15 期（1–3 月 6 半月报 + 4–12 月 9 月报）+ 状态开关
- 企业备案：13 字段全 SRS §4.5.1 校验，PENDING/APPROVED 状态机
- 调查期数据填报：状态机（DRAFT → 01 待市级审核），条件校验（调查期 < 建档期 ⇒ 减少类型/主因必填）
- 关闭期 / 未备案企业 拒绝填报
- 企业端 5 页：工作台 / 备案 / 填报 / 历史 / 改密
